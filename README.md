# Personal Portfolio Website

This is my **personal developer portfolio website**, originally built as part of the
[freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/).

After completing the certification requirements, I **further enhanced the project** with additional UI improvements, interactivity, and expanded sections to better reflect my personal brand as a frontend developer.

---

## 🚀 Live Demo

👉 https://tefol-hub.github.io/portfolio/

---

## 💼 Project Highlights

* Built a **fully responsive personal portfolio website**
* Includes structured sections:

  * Hero / Welcome section
  * About Me
  * Experience & certifications
  * Projects showcase
  * Contact information
* Implemented **both desktop and mobile navigation systems**

  * Standard navbar (desktop)
  * Hamburger menu (mobile)
* Integrated **interactive UI components using JavaScript**
* Showcases **real projects with live demos and GitHub links**
* Includes **social media integration (LinkedIn, GitHub, freeCodeCamp, etc.)**
* Enhanced beyond FCC requirements with:

  * Improved layout structure
  * Additional UI polish and animations
  * Expanded personal branding elements
